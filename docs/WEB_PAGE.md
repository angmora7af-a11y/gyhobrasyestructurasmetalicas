# Business Case: Sistema de Inteligencia de Mercado - G&H Bogotá

## 1. Resumen Ejecutivo
Implementación de un motor de extracción de datos automatizado para monitorear el catálogo de productos, precios y disponibilidad de G&H Bogotá, permitiendo análisis comparativos de mercado y optimización de inventario.

## 2. Objetivos
* **Análisis de Precios:** Obtener una visión en tiempo real de la competitividad de precios en el sector tecnológico.
* **Control de Stock:** Identificar brechas de disponibilidad en categorías clave (impresoras, periféricos, suministros).
* **Enriquecimiento de Catálogo:** Estructurar descripciones y atributos técnicos para estandarizar bases de datos internas.

## 3. Alcance Técnico
* **Fuente:** https://gyhbogota.com/
* **Frecuencia:** Ejecución semanal (Batch processing).
* **Formato de salida:** JSON / CSV para integración con BI.

## 4. Retorno de Inversión (ROI)
La automatización reduce en un **95%** las horas hombre dedicadas a la vigilancia manual de la competencia, permitiendo ajustes de precios dinámicos que pueden aumentar el margen de conversión en un **12%**.
